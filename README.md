# Air Quality Prediction

Predicting PM2.5 levels using machine learning.

## Project Structure

- `data/raw/` - Raw input data
- `data/processed/` - Cleaned/preprocessed data
- `models/` - Trained model files
- `notebooks/` - Jupyter notebooks for exploration
- `src/` - Source code (preprocess, train, evaluate, predict)
- `results/` - Output metrics and predictions

## Usage

1. Install dependencies: `pip install -r requirements.txt`
2. Preprocess data: `python src/preprocess.py`
3. Train model: `python src/train.py`
4. Evaluate: `python src/evaluate.py`
5. Predict: `python src/predict.py`
